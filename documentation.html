Requirements

  • Functioning game with A.I. and interactive board
  • User input button to restart or refresh the game
  • HTML5 audio clip of sonar sound
  • XML document loaded through the XMLHttpRequest

Software Design

Variables

  • Ships[], dead[], player[], computer[], playerships[], computerships[], shiptypes[].
  • rows (x), cols (y)
  • playerlives, computerlives, playflag

Functions

  • Model
  • setupPlayer()-Places ships on the grid.
  • onClick()-Handles the mouse clicking the board
  • computerMove()-The computer's logic in playing against the player.
  • View
  • init()-Starts all the scripts.
  • gameBoard()-Starts game.
  • imageLoad()-uploads all images in directory.
  • showGrid()-Creates game board.
  • setImage()-Changes image after clicked.
  • sinkShip()-Changes the entire image of a ship if sunk.
  • knowYourEnemy()-Displays computer's enemy ships when game is over.
  • getShips()-XML files that displays ship types and their size.
